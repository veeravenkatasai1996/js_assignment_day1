Methods of string

  • substring() - Extract the characters between two indices and return new substring.
  • splice()- Used to add or remove the elements from an array.
    Syntax: array.splice(index,howmanyitemstoberemoved,itemstobeadded)
  • replace()- Searches for the string of specified value and then replace the content.It will replace first occurence.
  • replaceAll() - Searches for the string of specified value and then replace the content.It will replace all occurences.
