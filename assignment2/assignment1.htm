The following are Methods of Console Function:

log(), error(), warn(), clear(), time() and timeEnd(), table(), count(), group() and groupEnd(), custom console logs.
console.log() -
Print the output to the console
Eg: console.log(1);

console.error() -
Used to log error message to console . By default it is highlighted in red color
Eg:console.error("Error in page");

console.warn() -
Used to log warning to the console. By default it is highlighted in yellow color
Eg:console.warn("Warning message");

console.clear() -
It is used to clear the console.
Eg:console.clear();

console.time() and console.timeEnd() -
Used to know amount of time spent on block or a function.
console.time(); console.timeEnd();

console.table() -
Used to generate a table inside console.
Eg:console.table({'a':1,'b':2})

console.count() -
Count the number the function hit.
Eg:console.count()

console.group() and console.groupEnd() -
Group into blocks. .

custom console logs -
Used for styling to custom logs . The syntax is to add %c parameter.
Eg:console.log("%c venkat",'color:blue')
